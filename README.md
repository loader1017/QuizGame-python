# QuizGame-python

## 개요
파이썬3을 활용한 터미널에서 작동 가능한 퀴즈 게임입니다.

## 퀴즈 주제 및 선정이유
초기 퀴즈 내용은 고양이입니다. 귀엽기 때문입니다.

## 실행 방법
윈도우에서는 터미널에 `python quizmain.py`
혹은 Mac Os 환경에서는 터미널에 `python3 quizmain.py`
입력 시 작동합니다.

## 기능 목록
1. 퀴즈 시작
2. 퀴즈추가 (내용 및 정답추가)
3. json에 저장된 퀴즈 목록 불러오기
4. json에 저장된 마지막 점수 불러오기
5. 프로그램 종료

## 파일구조, 설명
`quizmain.py`
퀴즈실행이 가능한 코드가 들어가있는 파일입니다.

`score.json`
퀴즈 실행 후 퀴즈를 풀면 마지막 점수가 저장되는 json 파일 입니다. 초기 설정은 null입니다.

`data.json`
퀴즈 문제와 정답이 들어가있는 json파일입니다. 초기 5개의 문제가 들어가있으며, 실행 후 퀴즈를 추가하면 반영,저장됩니다.

`Readme.md`
지금 읽고있는 이거입니다.
